# 💰 ExpenseIQ — Smart Expense Tracker

A modern, animated, and full-featured expense management application built using pure HTML, CSS, and Vanilla JavaScript. Designed for the Group 7 Web Technology Lab Project.

## ✨ Features

- **Dashboard**: Real-time spending overview, stats, and charts.
- **Transactions Management**: Add, edit, delete, and view a continuous list of transactions with robust group-filtering.
- **Reports & Analytics**: Insights into weekly, monthly, and yearly spending.
- **Budgeting**: Set limits on categories; get colorful and toast alerts as you near them.
- **Categories**: Dynamic category assignments with icons and custom colors.
- **Theme**: Premium 'Glassmorphism' dark mode UI, with seamless light-mode toggling.

## 🚀 How to Run

1. Open `index.html` in any modern web browser (Edge, Chrome, Firefox, Safari).
2. Enjoy the app! No backend server is required as everything is safely persisted to your browser's local storage.
3. *Tip: Head to **Settings** and click **Load Demo Data** to instantly populate the application with a rich sample dataset to explore the charts and functionalities!*
